# EV-0007: DP-0004 Core CI Validation Evidence

## Summary
- Adopted DP-0004 as the script-first validation workflow.
- Added a checked-in Rust toolchain file for Rust 1.87.0, `rustfmt`, `clippy`, and `wasm32-unknown-unknown`.
- Added top-level validation scripts for setup, version reporting, formatting, linting, tests, baseline builds, and full validation.
- Pinned the Rust toolchain to the exact `1.87.0` patch release and pinned `wasm-pack` setup to version `0.13.1`.
- Added a GitHub Actions validation workflow that calls repository scripts rather than duplicating validation commands.
- Replaced the deprecated async wasm-bindgen constructor with an async `WebGPU.create(...)` factory so Clippy can run with warnings denied.

## Commands Verified
- `scripts/develop`
- `scripts/version`
- `scripts/format --check`
- `scripts/lint`
- `scripts/test`
- `scripts/baseline`
- `scripts/validate`

## Result
- `scripts/develop` passed after installing or confirming Rust components, the WASM target, npm dependencies, and `wasm-pack`.
- `scripts/version` reported Rust 1.87.0, Cargo 1.87.0, rustfmt 1.8.0, Clippy 0.1.87, wasm-pack 0.13.1, Node 20.20.0, npm 10.8.2, and Vite 6.2.2 in the local environment.
- `scripts/format --check` passed.
- `scripts/lint` passed with `-D warnings`.
- `scripts/test` passed with 4 Rust tests.
- `scripts/baseline` passed, including `cargo check`, `wasm-pack build --target web`, and Vite production build.
- `scripts/validate` passed through the full version, format, lint, test, and baseline sequence.

## Notes
- In the Codex sandbox, `wasm-pack` needed escalated execution to write its normal `wasm-bindgen`/`wasm-opt` cache outside the repository. This is an environment permission issue, not a validation failure.
- `npm ci` reported existing npm audit findings: 1 moderate and 2 high vulnerabilities. DP-0004 explicitly defers npm audit policy and broader supply-chain gates to later work.
- Browser visual smoke validation remains deferred. The current baseline is build-level validation plus the existing browser demo instructions.
