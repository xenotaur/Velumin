# EV-0006: Glow Pipeline Spike Verification

## Summary
- Added a prototype glow pipeline on top of the thick-line primitive path.
- The renderer now draws widened vector geometry into a half-resolution offscreen bright-pass texture.
- Added a fullscreen composite shader that samples the offscreen texture with a small blur-like kernel.
- The final surface pass composites glow first, then draws the crisp core vector geometry on top.

## Spike Notes
- This is a proof of rendering flow, not final bloom quality.
- Current approach uses a half-resolution offscreen texture plus a 9-sample composite shader.
- Glow geometry is generated by scaling stroke width and intensity from the same command data as the crisp pass.
- The crisp line remains a separate final draw so the vector core stays sharp.

## Follow-up
- Tune glow radius, intensity, persistence, and background fade as production controls.
- Compare separable blur, mip/downsample chains, and multi-pass blur quality/performance.
- Move vector command and renderer internals into the planned core/renderer crate boundary before broadening public API.
